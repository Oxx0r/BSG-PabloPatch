# [cite_start]BSG-PabloPatch (Tabletop Simulator) 🎭 

Ein grafischer Mod-Patch für die **Battlestar Galactica** Implementation im Tabletop Simulator. [cite_start]Dieser Patch ersetzt diverse Texturen und Charakter-Gesichter durch das "Pablo"-Design. 

## ✨ Features
* [cite_start]**Custom TTS Textures:** Speziell optimierte Bilddateien für den Import in TTS. 
* [cite_start]**Pablo-fied Characters:** Ersetzt die Standard-Artworks der Charakterbögen und Karten durch Pablo-Varianten. 

## 🚀 Installation (TTS)
1. [cite_start]**Repository klonen oder ZIP laden:** Lade diesen Ordner herunter. 
2. [cite_start]**Dateien verschieben:** Kopiere die Bilder in deinen TTS-Cache oder lade sie direkt in deine Cloud hoch. 
   - [cite_start]Pfad meistens: `Documents\My Games\Tabletop Simulator`
